\subsection{Detailed Description}
Module\+: U\+S\+A\+RT Async Serial

Description\+: Provides asynchronous serial communication via any of the four U\+S\+A\+R\+TS available in the At\+Mega 2560.

Both interrupt driven and polling functions are available.

Uses free\+R\+T\+OS ring buffers to support interrupt functions.
